windows安装Linux子系统

发表于 2019-09-30 分类于 笔记 阅读次数：

首先打开Windows设置，选择更新与安全->开发者选项，打开开发人员模式

等待相关组件安装完成后，打开控制面板->程序和功能->启用或关闭Windows功能

勾选适用于Linux的Windows子系统

重启计算机

重启后前往Microsoft Store搜索相关应用或直接访问https://aka.ms/wslstore

安装子系统分发版

运行分发版之后，等待安装完成，输入用户名密码，即可使用

之后再PowerShell或cmd页面输入bash可以使用子系统

Windows文件系统在/mnt下，不同的盘符存放在对应的文件夹中
